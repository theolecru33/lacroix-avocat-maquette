# Lacroix Avocat — maquette one-page

Prototype de site vitrine pour **Maître Lionel Lacroix**, avocat en droit du travail
et de la sécurité sociale au Barreau de Bordeaux.

Réalisé par [BassinWebFlow](https://bassinwebflow.fr).

## Aperçu

- Page **statique** (HTML / CSS / JS), sans dépendance externe : polices `.woff2`
  et GSAP auto-hébergés, images en WebP (avec repli).
- Direction artistique : blanc, rouge bordeaux (#C1122B) et bleu marine (#15263E),
  titres *Prata*, texte *Mulish*.
- Responsive, accessible (WCAG 2.2), `noindex` (démo non indexée).

## Structure

```
index.html              page principale
mentions-legales.html   page légale (placeholder à compléter)
confidentialite.html    politique de confidentialité (placeholder)
css/                    style.css + responsive.css
js/                     interactions + animations GSAP (vendor auto-hébergé)
fonts/                  Prata + Mulish (.woff2)
img/                    logo, portrait, visuels (WebP + repli)
```

> Maquette de démonstration. Contenu, coordonnées et mentions légales
> restent à valider avec le client avant toute mise en ligne réelle.
